## Instalation

Download repo
``` git clone https://github.com/whos-gabi/iShop_crud_ops ```
<br>
``` cd iShop_crud_ops ```
<br>
Install libs
``` npm i ```
<br>
Create a ```.env``` file and fill in with your MongoDB credentials like in ```sample.env``` 
<br>
Run app
``` node index.js ```
